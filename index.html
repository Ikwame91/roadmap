Phase 03 · 10 weeks
Java & OOP Discipline
HTTP Server · Spring Boot · SOLID · Design Patterns · DSA
HTTP Server from Scratch
TCP · HTTP Parsing · State Machines · Threading
The project
Build a working HTTP/1.1 server from scratch in Java — no frameworks, raw TCP sockets
Learn Java through building: classes, interfaces, enums, generics, collections — all applied
TCP fundamentals — sockets, 3-way handshake, byte streams, chunked reads
HTTP request parsing — request line, headers, body using a state machine
HTTP response writing — status → headers → body enforced by a Writer state machine
Concurrency vs Parallelism — one thread per connection, why it doesn't scale, thread pools
SOLID in practice — Server handles connections, Handler handles routes (Single Responsibility)
Depth checkpoint
Why does TCP give bytes in chunks? How does your parser handle partial reads?
Trace the full journey: URL typed → DNS → TCP handshake → HTTP request → your server → response
Why does the Writer enforce status → headers → body order with a state machine?
Resource
Guide: github.com/viralcodex/http-server — ~500 lines total
Video: boot.dev HTTP server course (ThePrimeagen)
Mark complete
✓

Your Weekly Rhythm

Built around your real life — 7 to 5 job, long commute, 8–10 hrs/week available. Click each day to see what to do.

Mon – Thu
Review & Reinforce Only
~30 min
Re-read your notes from the weekend session — no new content
Small exercises only — one LeetCode easy (Phase 3+), or re-read one concept
Draw an architecture diagram from memory — no notes, no AI
The goal is consolidation, not progress. You're too drained for new learning after a 7–5.
Friday Eve
New Concept Session
2–3 hrs
You're decompressing from the week — use that energy before it turns into doomscrolling
Introduce one new topic — read, watch, understand the concept
Write down what confused you — these become your Saturday questions
Don't try to build yet. Friday is for understanding, Saturday is for building.
Saturday AM
Primary Deep Work Session — SACRED
3–4 hrs

Sunday PM
Apply & Consolidate
2–3 hrs
After church — afternoon is yours
Extend what you built Saturday, or tackle one thing that confused you
